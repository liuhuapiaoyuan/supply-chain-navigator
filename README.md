# Supply Chain Navigator (供应链中台大盘)

本项目是一个功能全面且易用的**供应链BI导航器/中台看板**系统，旨在为跨境电商及贸易型企业提供从生产采购到终端销售的全链路供应链监控、数据分析以及操作管理。系统基于 React + Vite 构建，结合现代化的可视化界面设计，极大地提升了供应链的数字化决策能力。

---

## 核心功能模块 (Core Features)

### 1. 首页仪表盘 (Dashboard)
提供全局视角的运营看板，多维度数据集中监控。
- **核心指标统计**：实时显示款式总数、SKU总数、FBA总库存、在途总量和工厂在产情况，并标注环比增减趋势。
- **精细化库存预警**：按 SKU 显示紧急（<7天）、警告（7-14天）的库存状态，同时下钻至具体的款号、可售天数以及智能给出的“补货建议”数据。
- **预警分布矩阵**：按款号汇总不同警戒级别的 SKU 数量，快速锁定风险品类。
- **待办与图表分析**：提供直观的待处理事项（To-do）列表、本月发货统计柱状图以及工厂订单分布饼图（Recharts驱动），辅以最近动态操作日志以便追溯。

### 2. SKU 快捷查询 (Quick Query)
聚焦单品维度的全链路追踪体系。
- **多条件智能搜索**：支持搜索指定 SKU，或通过“款式 + 颜色 + 尺码”交叉进行快速检索。
- **库存分布一站式汇总**：全景展示特定 SKU 在 **FBA库存**、**在途物流**、**工厂在产**、**本地仓库** 及预留状态下的精确数量。
- **健康度分析与明细跟进**：系统结合“日均销量”自动测算总可用库存支撑的“可售天数”。底部下放四大维度的明细列表（如仓储分布库位、发货批次追踪、工厂生产进度）。
- **趋势分析**：通过折线图呈现过去30天的库存动态起伏，辅助运营复盘判定。

### 3. 补货模拟器 (Simulation)
智能供应链决策辅助中心，进行动态库存推演。
- **库存健康矩阵**：二维矩阵式（颜色 VS 尺码）总揽选中款号之下所有 SKU 的可售天数及盈缺告警（红、黄、常规）。
- **动态沙盘推演**：支持针对各属性手动输入预估定采量，或一键自动“补货到60天 / 90天”进行系统推荐。
- **模拟前后对比**：沙盘计算出推演结果，直观对比模拟前后的“总库存量”和“预警SKU解除降比”。
- **操作闭环**：不仅作为沙箱演算，还支持保存预案、导出表格，以及无缝转化为正式的后续采购单。

### 4. 供应链台账管理 (Supply Chain Management)
分选项卡维护并推进生产与发运的核心执行层数据。
- **🚚 出库与发货管理**：管理FBA物流批次、承运方运费、目的仓；支持点击折叠面板按款号下钻查阅同批次发货明细，并可新增起草物流发运计划。
- **📋 采购订单跟进**：记录合同工厂分配情况、下单总量、已发货及剩余未交货数据；直观的进度条（Progress）展示跟踪交期状态。
- **🏭 供应商(工厂)档案**：统包合作工厂的业务规模与联系方式。并能下钻至“工厂详情页”，多维度追踪该工厂承接的每个属类款式的交付进度及在产金额，精细化压低生产逾期风险。

### 5. 操作审计日志 (Audit)
保障系统信息安全防篡改与流程可回溯能力。
- **完整动作追溯**：记录时间、操作人（系统或账户）、关联模块（出库/修改/商品管理）、具体对象与实施详情。
- **Diff式级联变更详情**：不仅记录“改了什么”，更针对特定修改动作（如调参、日期延迟、运费修改）收纳并对比“变更前 / 变更后”的数据差，并做色块区分。
- **复合过滤筛选**：结合自由文本搜索以及下拉分类（模块、操作类别），帮助管理员快速圈定异常操作来源。

## 技术特征摘要
- **前端框架**：`React` (Hooks & Router v6) + `TypeScript` 带来稳定且健壮的类型保障。
- **样式方案**：基于 `Tailwind CSS` 进行高度定制，并引入了一套深浅色适配（明亮/柔和主题）相关的 CSS 自定义变量变量（位于 `index.css`）。
- **页面交互UI**：采用现代化扁平化与圆角卡片（`stat-card`）组合，“类工作台 (Workbench)”排版提升业务人员长时操作体验；使用 `lucide-react` 提供优美一致的图标集。
- **可视化利器**：`Recharts` 组件强力驱动统计大屏及各类轻量级BI图表展示。
- **状态提示与全局遮罩**：提供了模态框隔离长表单输入，并配备 `sonner` / `toaster` 用于全局消息轻量推送。
